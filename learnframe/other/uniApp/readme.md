# uni app

`uni-app` 是一个使用 `Vue2` 开发跨平台应用的前端框架，开发者编写一套代码，可编译到`iOS`、`Android`、`H5`、小程序等多个平台。

快速体验

一套代码编到4个平台，难以置信吗？依次扫描4个二维码，亲自体验最全面的跨平台效果！
